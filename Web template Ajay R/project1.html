Design Statement:

When I first started this assignment, I did not have a firm idea of what I was going to make.
But, then I tried using a single item for the experiments. Like, for the neopixel experiments
I used a single neopixel with the example code. I then used the example code as a template
for my final code for the neopixel. I basically just make it more simplistic and it can change
between 3 lights.

For my motor experiments, I used the sweep example code to run the motor and I basically
experimented with various items most preferably items from my kit. In the final, I used a
piece of cardstock and had a wire hold it. I then made the wire attached to the plastic horn
of the motor. Allowing it to spin.
